## Python f-strings  
## Simple Explanation  
Les f-strings sont une façon simple d'insérer des variables dans des chaînes de caractères en Python. Elles permettent de créer des phrases dynamiques en combinant du texte et des valeurs variables. Cela rend le code plus clair et plus facile à lire. Elles sont disponibles depuis la version Python 3.6.  

## Key Concepts  
- Insertion de variables entre accolades `{}`  
- Valeurs dynamiques qui changent à chaque exécution  
- Chaînes formatées pour afficher des données  
- Syntaxe `f"{variable}"` pour envelopper le texte  
- Utilisation pour simplifier les messages ou calculs  

## Example  
```python  
nom = "Alice"  
age = 25  
print(f"{nom} a {age} ans.")  
```  
Ce code affiche : *Alice a 25 ans.*
